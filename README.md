# 📚 Abrehot Book Club

A modern, cozy web application for managing a book club, designed to feel like a high-end independent bookstore meets a modern social app.

![Abrehot Book Club](https://images.unsplash.com/photo-1524995997946-a1c2e315a42f?w=1200&h=400&fit=crop)

## ✨ Features

- 📖 **Book Library** - Browse past and future book selections with ratings
- 💬 **Discussion Boards** - Threaded comments for each book
- 📅 **Meeting Scheduler** - Calendar view for upcoming meetings
- 👥 **Member Directory** - Connect with fellow readers
- 🎨 **Warm Design** - Cream, espresso, and terracotta color palette
- 📱 **Responsive** - Works beautifully on all devices
